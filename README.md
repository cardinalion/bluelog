# Bluelog

*A blue blog.*

Demo: http://bluelog.helloflask.com

![Screenshot](http://helloflask.com/screenshots/bluelog.png)

## Installation

```
$ git clone https://github.com/greyli/bluelog.git
$ cd bluelog
$ pipenv install --dev
$ pipenv shell
$ flask forge
$ flask run
* Running on http://127.0.0.1:5000/
```

Test account:

* username: `admin`
* password: `helloflask`
